# Webpack Template

Webpack 5 starter template with support for:

- JavaScript (ES6+) via Babel
- CSS
- Images (png, jpg, jpeg, gif, svg, webp)
- Live reload (HMR)

## How to use this template

1. Click **"Use this template"** → **"Create a new repository"**
2. Clone the new repo locally:
```bash
   git clone REPO_URL
   cd REPO_NAME
```
3. Install dependencies:
```bash
   npm install
```

## Available commands

| Command | Description |
|---------|-------------|
| `npm start` | Starts the development server at `http://localhost:3000` |
| `npm run build` | Production build in `dist/` folder |
| `npm run build:dev` | Development build in `dist/` folder |
